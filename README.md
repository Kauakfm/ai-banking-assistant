# AI Banking Assistant

Um assistente inteligente para operações bancárias alimentado por IA.

## Estrutura do Projeto

Este é um monorepo contendo os seguintes componentes:

- **bfa-go**: Backend principal em Go
- **agent-service**: Serviço de agente de IA
- **knowledge-base**: Base de conhecimento e recuperação de informações
- **docs**: Documentação do projeto
- **scripts**: Scripts de utilitários e deploy
- **tests**: Testes integrados

## Pré-requisitos

- Docker e Docker Compose
- Go 1.21+
- Node.js 18+ (se aplicável)

## Iniciando

```bash
docker-compose up
```
